## Playwright Tests 2026-06-14: new specs 19–23

This session added end-to-end coverage for accessibility, the expanded Rules
feature, imports, and previously-untested documented features, plus broader REST
API coverage.

## New specs

| Spec | Area | What it covers |
|------|------|----------------|
| 19 | Accessibility | Page language, skip-to-content link, landmark roles, visible focus, dialog roles, accessible names on icon controls, no duplicate element ids |
| 20 | Rules (automation) | Fullscreen Rules page, creating an event rule, Import/Export dialog (JSON/CSV), selecting rules, the visual Workflow view, and a board button rendered in the board header |
| 21 | Import without mapping | The "Import without mapping members (map later)" button skips the member-mapping step and imports a WeKan board immediately |
| 22 | Card & list features | Stickers, card locations, the minicard complete checkbox, and enabling a WIP limit on a list |
| 23 | REST API (more) | Rules API (create/list/get/edit/delete), the new card `stickers` / `locations` / `dueComplete` fields, whole-board import (`POST /api/boards/import` recreates a board with its rules/workflows), and core CRUD for swimlanes, lists, custom fields, checklists + items and comments |
| 24 | Feature issues | #459 accessible reordering (move card down / move list right), #3984 visual card aging (default + configurable thresholds), #4266 board delete removes rules/triggers/actions, #5157 board background on the All Boards tile |
| 25 | Excel & PDF | #395 Excel (.xlsx) board import (fixture built in-test with exceljs) and board PDF export over REST |

## Notes

- Specs 17 and 23 require the API server (`WITH_API=true`), which the
  `rebuild-wekan.sh` "Run tests" server sets. They authenticate with a real Bearer
  token (the seeded user's MongoDB resume token) and verify the resulting MongoDB
  changes directly.
- The Rules page is a board-scoped route (`/b/:id/:slug/rules`); spec 20 navigates
  to it directly to avoid fragile menu navigation.
- Run a single spec with:
  `cd tests/playwright && npx playwright test specs/20-rules.e2e.js`
