# Dev: KaiOS install & update platforms (manual vs automatic, like Snap)

**KaiStore** is the sole store for these HTML5 feature-phone devices; the store client
self-updates silently in the background, and app updates are store-managed over the
Internet.

| Store/Mechanism | Manual | Automatic | Auto/Internet mechanism | Maintainer | URL | License |
|---|:---:|:---:|---|---|---|---|
| **KaiStore** | ✓ | ✓ | Background app updates; store client self-updates silently | KaiOS Technologies | https://www.kaiostech.com/store/ | Proprietary |

## WeKan applicability
**Not viable** — WeKan's UI is unsuited to feature-phone HTML5 constraints; no realistic
path.
