Official Status and Private Pre-paid Priority Support is at email, not at GitHub

Official Status is only available by emailing support@wekan.fi

Public GitHub issues are not a support area. GitHub issues do not have any priority. GitHub issues are only for FLOSS developers, that are able to write detailed anonymized details and screenshots about bugs, add to issues more details and clues when bug happens, or able send PR to fix some GitHub issue. GitHub issues are also full of newbie trolls and flamewars, those that have not supported WeKan in any way, not donated, not sent any PR, so if you have paid something for WeKan, please do not participate to GitHub issues, so that you would not be categorized as FLOSS developer or troll, instead email support@wekan.fi . Answers at GitHub issues are not official status, they are incomplete outdated discussions like Twitter or Facebook.

All paid features are being developed and added to WeKan, there is new releases with major features and fixes about every 3rd day since December 2016. WeKan already has new features other kanban software does not have. Security issues have been fixed immediately. Upgrade info is at https://wekan.fi/upgrade/. If there is any problems, please email support@wekan.fi
